Code Snippet

            const int sensorPin = A0; //set sensor pin
            const int ledPin = 9; //set led pin 1
            const int ledPin2 = 5; //set led pin 2
            
            int sensorValue, high = 0, low = 1023; //setting the high and low of the photo sensor (given values)
            
            void setup() { //setup
              Serial.begin(9600); //starting serial print
              pinMode(ledPin, OUTPUT); //set output for first LED pin
              pinMode(ledPin2, OUTPUT); //set output for second LED pin
            }
            
            void loop() { //loop to repeat process
              sensorValue = analogRead(sensorPin); //read current level of light from sensor
            
                Serial.print("Input (sensor) ="); //printing out input, based on how much light read from sensor
                Serial.println(sensorValue); //print value
            
              //high and low might not be 0 and 1023, so we are finding new high and low values
              if (sensorValue < low) { //if statement finds low value
                low = sensorValue;  //sets a low value
              }
              if (sensorValue > high) { //if statement finds high value 
                high = sensorValue; //sets a low value
              }
            
              //
              sensorValue = map(sensorValue, low+5, high-5, 0, 255); 
              //maps the sensor value from a range from 0-1023 to 0-255, but needs an offset to make sure fully turns on and off
              sensorValue = constrain(sensorValue, 0, 255); //constrains the value to the limits 
            
            
                analogWrite(ledPin, sensorValue); //write the new value to the led based on the sensor
                analogWrite(ledPin2, sensorValue); //write the new value to the led based on the sensor
            
                Serial.print("\t Output (LED light) =");//printing out output, based on how much light the led uses 
                Serial.println(sensorValue); //print value
            }

Schematics and Calculations

The above image displays schematics for the photoresistor and two LEDs respectively. The photoresistor is followed by a 10K resistor, shown in the explanation and calculations below.

This image is an explanation of the kinds of inputs you might read at the voltage divider through the sensor. The voltage divider reads around 540 when the photosensor is exposed to a moderate amount of light, which is converted as an output to 255 to the LED, which is the brightest it gets. The picture further explains this idea, as well as why we use 10K ohm resistors for the photoresistor.
